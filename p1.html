Project 1: Personal Portfolio Website

A detailed guide to building your first major project using HTML and CSS.

Introduction

Project Goal: To build a clean, professional, and responsive single-page portfolio website from scratch. This will be your digital resume and the place to showcase all the amazing projects you'll build in the future.

Why it's important: A portfolio is the most important asset for a web developer. It proves your skills to potential employers far better than a resume alone. This foundational project teaches you the essential structure and styling of a modern webpage.

Technologies Used: We'll stick to the fundamentals: HTML5 for structure, and CSS3 for styling (including modern layout techniques like Flexbox and Grid). We will also use Git for version control.

Final Product Preview

Here is a simple preview of what we're going to build. It's designed to be clean, modern, and easy for recruiters to navigate.

Step 1: The HTML Structure

First, create a new folder for your project. Inside it, create a file named index.html. This file will contain the entire structure of our website. We'll use semantic HTML tags like <header>, <section>, and <footer> to make our code readable and accessible.

Copy and paste the code below into your index.html file.

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Your Name - Web Developer</title>
    <link rel="stylesheet" href="style.css">
</head>
<body>

    <!-- Header and Navigation -->
    <header>
        <nav>
            <div class="logo">Your Name</div>
            <ul>
                <li><a href="#about">About</a></li>
                <li><a href="#projects">Projects</a></li>
                <li><a href="#contact">Contact</a></li>
            </ul>
        </nav>
    </header>

    <main>
        <!-- Hero Section -->
        <section id="hero">
            <h1>Aspiring Full-Stack Developer</h1>
            <p>I build beautiful and functional websites.</p>
            <a href="#projects" class="cta-button">View My Work</a>
        </section>

        <!-- About Section -->
        <section id="about">
            <h2>About Me</h2>
            <div class="about-container">
                <p>Hello! I'm a passionate developer learning to build amazing things for the web. I have a strong foundation in HTML, CSS, and JavaScript, and I'm currently diving deep into the MERN stack. I love solving problems and creating intuitive, user-friendly experiences.</p>
            </div>
        </section>

        <!-- Projects Section -->
        <section id="projects">
            <h2>My Projects</h2>
            <div class="projects-grid">
                <!-- Project Card 1 -->
                <div class="project-card">
                    <h3>Project Title 1</h3>
                    <p>A brief description of your project will go here. Talk about the technologies used and the problem it solves.</p>
                </div>
                <!-- Project Card 2 -->
                <div class="project-card">
                    <h3>Project Title 2</h3>
                    <p>This is where you'll showcase your future projects from this roadmap!</p>
                </div>
            </div>
        </section>

        <!-- Contact Section -->
        <section id="contact">
            <h2>Get In Touch</h2>
            <p>I'm always open to discussing new projects or opportunities.</p>
            <a href="mailto:your.email@example.com" class="cta-button">Email Me</a>
        </section>
    </main>

    <!-- Footer -->
    <footer>
        <p>© 2025 Your Name. All rights reserved.</p>
    </footer>

</body>
</html>

Step 3: Version Control & Deployment

Now that you've built your site, it's time to put it on the internet for everyone to see. We'll use Git and GitHub to save our code and Netlify to host it for free.

1

Initialize Git

Open your terminal or command prompt, navigate to your project folder, and run this command to turn it into a Git repository.

git init
2

Commit Your Code

Add your files to Git and make your first "commit," which is like a snapshot of your code.

git add .
git commit -m "Initial commit: Create portfolio website"
3

Push to GitHub

Go to GitHub, create a new repository (don't initialize it with a README). GitHub will give you commands to connect your local repository and "push" your code to it. They will look like this:

git remote add origin https://github.com/your-username/your-repo-name.git
git branch -M main
git push -u origin main
4

Deploy with Netlify

This is the easiest part. Go to Netlify and sign up with your GitHub account. Find the option to deploy a new site from Git, connect your GitHub, and select the portfolio repository you just created. Netlify will automatically build and deploy your site to a live URL!
